# Guilherme's Machine 1.0.0 🖥️

O **Guilherme's Machine** é um conversor de bases numéricas interativo desenvolvido para facilitar o entendimento de sistemas de numeração computacionais. Este projeto foi construído utilizando tecnologias web fundamentais como parte dos meus estudos em Ciência da Computação na **SPTech**.

## 🚀 Funcionalidades

A aplicação permite a conversão entre as seguintes bases:
* **Decimal**: Base 10.
* **Binário**: Base 2.
* **Hexadecimal**: Base 16.
* **Octal**: Base 8.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação dos elementos de seleção e entrada de dados.
* **CSS3**: Design moderno com uso de gradientes e efeitos de transparência.
* **JavaScript (Vanilla)**: Manipulação dinâmica do DOM e lógica de conversão através dos métodos `parseInt()` e `toString()`.

## 🎨 Design e Interface

O projeto conta com uma interface estilizada para oferecer uma boa experiência ao usuário:
* **Paleta de Cores**: Fundo em tons de azul profundo com destaques em ciano para os resultados.
* **Interatividade**: Botões com transições suaves e efeito de escala ao passar o mouse.


## 📖 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/GuiPastorello/Guilherme-Machine-1.0.0.git

2. Abra o arquivo GuilhermeMachine.html em qualquer navegador moderno.

3. Selecione a base de origem, insira o valor e clique em Converter.
